# To Run Locally
## Clone the project
```bash
  git clone https://github.com/nikhildeora/cool-monk-ecommerce.git
```

Go to the project directory

```bash
  cd cool-monk-ecommerce

```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run start
```
