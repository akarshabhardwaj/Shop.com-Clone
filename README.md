# To Run Locally
## Clone the project
```bash
  git clone https://github.com/nikhildeora/delightful-secretary-78.git
```

Go to the project directory

```bash
  cd delightful-secretary-78

```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run start
```
